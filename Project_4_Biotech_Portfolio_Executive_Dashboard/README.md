# Biotech Portfolio Executive Dashboard

## R&D Portfolio Risk, Value & Investment Strategy

## Overview
This project simulates an executive-level biotech R&D portfolio dashboard designed to support strategic decision-making across pipeline risk, value, budget allocation, and development timelines.

The dashboard enables portfolio leaders to assess investment efficiency, risk exposure, and program prioritization.

## Objectives
- Provide executive KPI visibility across the R&D portfolio
- Evaluate risk versus expected value across programs
- Analyze budget allocation by development phase
- Assess timeline risk and long-term exposure

## Key Features
- Executive KPI cards (Programs, Budget, Revenue, Success Rate)
- Risk vs Value portfolio map
- Budget allocation by development phase
- Program distribution by therapeutic area
- Timeline risk map with threshold indicators

## Data
- Simulated biotech portfolio dataset for demonstration purposes

## Tools Used
- Tableau Public
- Excel (data modeling)
